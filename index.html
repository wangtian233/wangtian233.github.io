hexo+github搭建个人博客

hexo是目前最主流的静态博客框架，该框架基于node，因此编译效率非常高，同时借助github可以实现零成本搭建和维护。

环境搭建

安装node环境

在node官网下载对应操作系统的node安装包，安装过程很无脑，就是next模式，安装完毕之后通过以下步骤检测是否安装成功：

  • win+r打开命令行窗口
  • 输入cmd回车
  • 在cmd窗口输入node -v

如果提示“node不是外部或内部命令”，则表示安装失败。

安装git

git: 版本控制器
在git官网下载对应操作系统的git安装包，next模式安装

安装vscode编译器

vscode是一款强大的IDE

安装hexo脚手架

1.win+r输入cmd回车
2.输入如下命令回车

  • npm:依赖包管理器
  • install：安装、导入，可以简写为i
  • -g：代表全局安装

笔记

hexo+node+git搭建个人博客

环境、工具的安装

node：一款牛叉的web服务器，目前很多大型项目都使用node搭建

特点：解析效率高，没有兼容性
比较适合于io密集型的项目
版本：建议版本6.10（比较稳定，兼容性好）
如何安装：下载相应操作系统的安装包
如何测试：
1.win+r输入cmd回车
2.在新打开的命令行输入node -v
3.如果能显示node版本则安装成功，否则重新安装

名称 含义
vscode 多语言开发工具
git 版本控制器

安装hexo脚手架

1.win+r输入cmd回车
2.在cmd窗口输入npm install -g hexo-cli

名称 含义
npm 依赖包管理工具
Install 导入/安装
-g 全局安装

初始化一个博客项目

hexo init 项目名称（英文）
如何建立文件：在主目录，鼠标右键选择git bash here，输入命令hexo init 项目名称（英文）
init：初始化、

进入项目，添加依赖包

1.进入到项目主目录文件夹
2.鼠标右键选择git bash here
3.使用npm install命令添加依赖包
目的：讲项目所需依赖包添加到本地

启动服务器

1.在博客目录右键git bash here
2.输入hexo server
3.在浏览器地址栏输入：localhost:4000回车预览

发表文章

hexo new 文件名

博客预览

1.Ctrl+s保存
2.hexo g编译
3.回到浏览器Ctrl+r刷新
