hexo+github搭建个人博客

hexo是目前最主流的静态博客框架，该框架基于node，因此编译效率非常高，同时借助github可以实现零成本搭建和维护。

环境搭建

安装node环境

在node官网下载对应操作系统的node安装包，安装过程很无脑，就是next模式，安装完毕之后通过以下步骤检测是否安装成功：

  • win+r打开命令行窗口
  • 输入cmd回车
  • 在cmd窗口输入node -v

如果提示“node不是外部或内部命令”，则表示安装失败。

安装git

git: 版本控制器
在git官网下载对应操作系统的git安装包，next模式安装

安装vscode编译器

vscode是一款强大的IDE

安装hexo脚手架

1.win+r输入cmd回车
2.输入如下命令回车

  • npm:依赖包管理器
  • install：安装、导入，可以简写为i
  • -g：代表全局安装

个人域名绑定

1.域名申请（自行百度）
2.域名解析，添加如下记录值
记录类型|主机记录|解析线路（isp）|记录值
-|-|-|-
CNAME|www|默认|用户名.github.io
A|@|默认|192.30.252.153
A|@|默认|192.30.252.154
3.在博客主目录下找到public文件夹
在里面新建名为CNAME的文件（不带后缀）
4.在CNAME文件里面填写个人域名，如baidu.com
5.执行hexo d命令推送
